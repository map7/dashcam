* Dashcam

Used for either my Car or Bike to loop record.

https://github.com/map7/dashcam

** Hardware

- Raspberry PI 4
- PI Camera V1
- Sdcard 64GB Sandisk Endurance
- OnOffshim
- 2 x Buttons & LED

** Usage

- Boot
- Hit the record button, LED should start
- Videos are kept in /home/pi/dashcam/videos
